# Менеджер нейросетей v3.0

Приложение для отправки вопросов к нейросетям и сохранения ответов.

## Установка

1. Установите Python 3.8+
2. Скопируйте все файлы в одну папку
3. Запустите `install.bat` для установки зависимостей

## Настройка API ключей

1. Откройте приложение (`python main_app.py` или `run.bat`)
2. Перейдите на вкладку "Настройки API"
3. Введите API ключи для нужных нейросетей
4. Настройте Telegram бота (если требуется)
5. Нажмите "Сохранить конфигурацию"

### Получение API ключей

- **OpenAI**: https://platform.openai.com/api-keys
- **Anthropic**: https://console.anthropic.com/
- **DeepSeek**: https://platform.deepseek.com/api_keys
- **Telegram Bot**: Создайте через @BotFather в Telegram

## Использование

1. Создайте текстовый файл с вопросом
2. Выберите файл через интерфейс
3. Выберите папку для сохранения ответов
4. Отметьте нужные нейросети
5. Нажмите "Отправить запрос"

## Отправка в Telegram

1. Введите токен бота и Chat ID
2. Файлы автоматически отправляются после сохранения
3. Или отправьте вручную через вкладку "История"

## Особенности приложения

✅ **Поддержка 3 нейросетей**: OpenAI GPT, Anthropic Claude, DeepSeek  
✅ **Графический интерфейс**: Удобный интерфейс на tkinter  
✅ **Сохранение конфигурации**: Все настройки сохраняются автоматически  
✅ **Логирование**: Подробный лог выполнения операций  
✅ **История запросов**: Просмотр и управление предыдущими запросами  
✅ **Отправка в Telegram**: Автоматическая отправка результатов  
✅ **Безопасность**: API ключи скрыты при вводе  
✅ **Многопоточность**: Не блокирует интерфейс при выполнении запросов  
✅ **Проверка соединения**: Визуальные индикаторы статуса подключения  
✅ **Уникальные имена файлов**: Автоматическое предотвращение конфликтов имен  

## Структура проекта

- `main_app.py` - основной файл приложения
- `azure.tcl` - файл стилей интерфейса
- `config.json` - файл конфигурации
- `requirements.txt` - зависимости Python
- `install.bat` - установочный скрипт
- `run.bat` - скрипт запуска
