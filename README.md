__Инструкция по использованию:__
*Установка:*

Установите Python 3.8+

Скопируйте все файлы в одну папку

Запустите install.bat для установки зависимостей

__Настройка API ключей:__

Откройте приложение (python main_app.py или run.bat)

Перейдите на вкладку "Настройки API"

Введите API ключи для нужных нейросетей

Настройте Telegram бота (если требуется)

Нажмите "Сохранить конфигурацию"

___Получение API ключей:___

OpenAI: https://platform.openai.com/api-keys

Anthropic: https://console.anthropic.com/

Google AI: https://makersuite.google.com/app/apikey

YandexGPT: https://console.cloud.yandex.ru/

Cohere: https://dashboard.cohere.com/api-keys

__Telegram Bot: Создайте через @BotFather в Telegram__

Использование:

Создайте текстовый файл с вопросом

Выберите файл через интерфейс

Выберите папку для сохранения ответов

Отметьте нужные нейросети

Нажмите "Отправить запрос"

Отправка в Telegram:

Введите токен бота и Chat ID

Файлы автоматически отправляются после сохранения

Или отправьте вручную через вкладку "История"

___Особенности приложения:___
✅ Поддержка 5 нейросетей: OpenAI, Anthropic, Google, Yandex, Cohere
✅ Графический интерфейс: Удобный интерфейс на tkinter
✅ Сохранение конфигурации: Все настройки сохраняются автоматически
✅ Логирование: Подробный лог выполнения операций
✅ История запросов: Просмотр и управление предыдущими запросами
✅ Отправка в Telegram: Автоматическая отправка результатов
✅ Безопасность: API ключи скрыты при вводе
✅ Многопоточность: Не блокирует интерфейс при выполнении запросов
